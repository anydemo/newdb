new db write by golang. Support mysql TIBD sql

## Getting Started

### Prerequisites

see `go.mod`

### Installing

`Makefile` will tell you how to install it and use it

run one demo

```
make help
make demo
```

## Running the tests

```
make test-verbose test-race test-coverage test
```
